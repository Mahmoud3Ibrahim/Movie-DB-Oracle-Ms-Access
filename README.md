# Enterprise Hybrid Data Architecture: Oracle-to-Access Integration
**Advanced Connectivity & Relational Systems Engineering**

---

## Project Purpose
This project was engineered to solve the common enterprise challenge of data silos. By architecting a synchronized environment between an **Oracle SQL Server** backend and a **Microsoft Access** frontend, I established a "Single Source of Truth" that allows for high-stakes analytical reporting without the risks of manual data duplication.

---

## Core Technical Workflows

### 1. Backend Engineering & Data Modeling (Oracle SQL)
* **Relational Schema Design:** Architected comprehensive database schemas from the ground up, ensuring strict 3rd Normal Form (3NF) compliance to eliminate data redundancy.
* **Database Administration (DBA):** Managed the full security lifecycle, including user creation, role-based privilege management (GRANT/REVOKE), and account authentication protocols.
* **SQL Scripting:** Developed and executed optimized SQL scripts to build tables, define constraints, and populate large-scale datasets with referential integrity.

### 2. Middleware & Enterprise Connectivity (ODBC)
* **Connectivity Layer:** Configured the **ODBC (Open Database Connectivity)** bridge to enable bi-directional communication between the server and the client interface.
* **Live Table Linking:** Implemented DSN (Data Source Name) configurations to link Oracle backend tables as live entities within the Access environment, ensuring real-time data synchronization.
* **Performance Tuning:** Optimized connection strings and query execution paths to maintain system stability during high-volume data retrieval.

### 3. Frontend Development & Analytical UX (MS Access)
* **Integrated Reporting Interface:** Built a robust user interface featuring automated data-entry forms and complex queries that interact directly with the Oracle engine.
* **Advanced Querying:** Designed multi-layer queries to transform raw backend data into actionable insights, mirroring the reporting needs of large-scale infrastructure and planning agencies.
